# Империя чатов

VK-бот: беседа = государство. Работы, казна, рейды, арсенал, лавка, сезоны.

## Запуск

```bash
copy .env.example .env
# VK_TOKEN + GROUP_ID=240303101
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**Играй в ЛС с ботом** — в беседе reply-клавиатура общая на всех.

## Команды

| Кнопка | Действие |
|--------|----------|
| Ежедневка | Стрик крон (МСК) |
| Работа | 7 работ + контрабанда (свои КД) |
| Страна | Основать / вступить / инфо |
| Казна | Траты лидера/казначея, щит |
| Цель недели | Общая цель страны |
| Роли | Воевода, казначей, глашатай |
| Лавка | Выкуп, энергия, удача, колесо |
| Сумка | Экип, заточка, кодекс |
| Война / Рейд | Шанс от активных граждан |
| Сезон | Месячный рейтинг стран |
| Дуэль / Торг / Аукцион | PvP и рынок |
| Предложение | Идея обновления (награда при принятии) |
| Как играть | Полный гайд |

## Экономика

- Старт: 100 крон, энергия до 8
- Ежедневка: 40 + 10×стрик (кап 7)
- Работы: таверна ~10м … охрана ~40м; налог → казна
- Лавка: выкуп из тюрьмы, эликсир, печать удачи, вклад, знамя, колесо
- Казна: указ (+работы), военный сбор, раздача, амнистия, щит 24ч
- Рейд: сила = граждане (активные 48ч важнее) + экип; можно отбить
