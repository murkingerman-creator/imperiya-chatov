# Империя чатов

VK-бот: беседа = государство. Работы с мини-играми, кастомизация стран, инвайты, хроника.

## Запуск

```bash
copy .env.example .env
# VK_TOKEN + GROUP_ID=240303101
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Команды

| Кнопка | Действие |
|--------|----------|
| Ежедневка | Стрик крон (МСК) |
| Работа | Шахта / Рынок / Охрана + мини-игра |
| Страна | Основать, вступить, инфо, выйти |
| Оформить | Флаг, герб, девиз, налог… (лидер) |
| Трон | Передать лидерство |
| Распустить | Удалить страну (лидер, для тестов) |
| Инвайт | Код друга → бонусы |
| Война | Рейд лидера |
| Топы | Страны / игроки |

## Экономика

- Ежедневка: 40 + 10×стрик (кап 7)
- Шахта 90м / Рынок 45м / Охрана 120м
- Смена страны: кулдаун 24ч
- Инвайт: +80 / +40 / +50 казне
- Хроника: пост на стену группы ~20:00 МСК

## Распустить страну

Лидер → Страна → Распустить → подтвердить. Беседу можно основать заново.
