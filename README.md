# Империя чатов

VK-бот: беседа = государство. Работы, казна, рейды, арсенал, лавка, сезоны.

## Запуск

```bash
copy .env.example .env
# VK_TOKEN + GROUP_ID=240303101
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**Играй в ЛС с ботом** — в беседе reply-клавиатура общая на всех.

## Команды

| Кнопка | Действие |
|--------|----------|
| Ежедневка | Стрик крон (МСК) |
| Работа | 7 работ + контрабанда (свои КД) |
| Страна | Основать / вступить / инфо |
| Казна | Траты лидера/казначея, щит |
| Цель недели | Общая цель страны |
| Роли | Воевода, казначей, глашатай |
| Союз | Альянс двух стран: сила в рейде + доля добычи |
| Лавка | Выкуп, энергия, удача, колесо |
| Сумка | Экип, заточка, кодекс |
| Война / Рейд | Шанс от активных граждан |
| Сезон | Месячный рейтинг стран |
| Дуэль / Торг / Аукцион | PvP и рынок |
| Предложение | Идея обновления (награда при принятии) |
| Баг | Сообщить об ошибке (награда при подтверждении) |
| Как играть | Полный гайд |

## Экономика

- Старт: 100 крон, энергия до 8
- Ежедневка: 40 + 10×стрик (кап 7)
- Работы: таверна ~10м … охрана ~40м; налог → казна
- Лавка: выкуп из тюрьмы, эликсир, печать удачи, вклад, знамя, колесо
- Казна: указ (+работы), военный сбор, раздача, амнистия, щит 24ч
- Рейд: сила = граждане (активные 48ч важнее) + экип + ~55% силы союзника; можно отбить
- Союз: 1 на страну; лидеры предлагают/принимают; рейд по союзнику запрещён; 25% добычи союзу

## Админка

VK id `525336510`, `784179630`. Только в ЛС → Админ.

- `!дать ID СУММА` · `!игрок ID` · `!кд ID` · `!энергия ID`
- `!удалитьстрану Название`
- `!объявление текст` — во все беседы и ЛС
- кнопки: 📣✉️ Везде / 🎁 Всем кроны / 🌤 Ивенты / 🎮 Ещё
- `💡 Предложения` · `!принять ID` · `!отклонить ID [причина]`
- `🐛 Баги` · `!багпринять ID` · `!баготклонить ID [причина]`
- `!всем СУММА [текст]` — бонус крон всем + рассылка в беседы и ЛС
- **Ивенты:** `!ивент KEY [часы]` · `!стоп_ивент` (урожай, мир, дождь лута…)
- **Вспышки:** авто раз в 2–3ч (30 штук) · `!вспышка` / `!вспышка KEY` · `!вспышка_стоп`
- **Ещё:** тюрьма/свобода, предмет, титул, джекпот, дождь стране, топ, поиск
