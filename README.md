# Империя чатов — MVP

VK-бот: **беседа = государство**. Работай, копи кроны, наполняй казну, устраивай рейды.

## Быстрый старт

1. Скопируй конфиг и вставь токен группы:

```bash
copy .env.example .env
```

Отредактируй `.env`:

```
VK_TOKEN=vk1.a....твой_ключ
GROUP_ID=240303101
```

2. Установи зависимости:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

3. Запусти:

```bash
python main.py
```

4. В VK: напиши сообществу **Старт** → добавь сообщество в беседу → **Страна** → **Основать**.

## Команды

| Кнопка / текст | Действие |
|----------------|----------|
| Старт / Меню | Главное меню |
| Профиль | Кроны, энергия, страна |
| Работа | +30–60 крон (раз в час, −1 энергия) |
| Страна | Инфо / основать / вступить |
| Война / рейд | Рейд лидера на другую страну |
| Топ стран / Топ игроков | Рейтинги |

## Экономика MVP

- Старт: 100 крон, энергия 5
- Энергия: +1 каждые 20 мин
- Налог: 10% с работы в казну страны
- Основание страны: 50 крон (только в беседе)
- Рейд: кулдаун 3 ч, 5–15% казны цели

## Структура

- `main.py` — запуск Long Poll
- `bot/` — конфиг, клавиатуры
- `db/` — SQLite модели
- `services/` — игровая логика
- `handlers/` — команды VK

База: `data/empire.db` (создаётся автоматически).
