# Империя чатов

VK-бот: беседа = государство. Работы, казна, рейды, арсенал, лавка, сезоны.

## Запуск

```bash
copy .env.example .env
# VK_TOKEN + GROUP_ID=240303101
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**Играй в ЛС с ботом** — в беседе reply-клавиатура общая на всех.

## Команды

| Кнопка | Действие |
|--------|----------|
| Ежедневка | Стрик крон (МСК) |
| Работа | Уровни открывают работы · мини-игры |
| Казна | Указы, праздник, укрепление, стипендия, монумент, фонд рейда |
| Страна | Основать / вступить / инфо |
| Казна | Траты лидера/казначея, щит |
| Цель недели | Общая цель страны |
| Роли | Воевода, казначей, глашатай |
| Союз | Альянс двух стран: сила в рейде + доля добычи; разрыв = штраф |
| Районы | Рынок / казарма / храм за казну |
| Осада / Сбор | Мульти-рейд и сбор граждан |
| Континент | Север/Юг/Центр — недельная война блоков |
| Сага / Контракты | Личная цепочка и заказы страны |
| Лавка | Выкуп, энергия, удача, колесо |
| Сумка | Экип, заточка, кодекс |
| Война / Рейд | Шанс от активных граждан |
| Сезон | Месячный рейтинг стран |
| Дуэль / Торг / Аукцион | PvP и рынок |
| Предложение | Идея обновления (награда при принятии) |
| Баг | Сообщить об ошибке (награда при подтверждении) |
| Как играть | Полный гайд |

## Экономика

- Старт: 100 крон, энергия до 8, уровень 1
- Уровни: XP с работ/ежедневки/рейдов; работы и фичи открываются по уровню
- Ежедневка: 40 + 10×стрик (кап 7)
- Работы: таверна/рыбалка с 1 ур. … охрана с 6, контрабанда с 8
- Казна: указ, сбор, раздача, амнистия, праздник, укрепление, стипендия XP, фонд рейда, монумент, щит
- Рейд: сила = граждане + сбор + экип + казарма + союз; укрепление страны защищает

- **Ивенты:** `!ивент KEY [часы]` · `!стоп_ивент` (урожай, мир, дождь лута…)

- **Ещё:** тюрьма/свобода, предмет, титул, джекпот, дождь стране, топ, поиск
